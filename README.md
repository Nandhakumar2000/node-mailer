# Node.js send email tutorial.

## Requirements

* Nodejs 8.10.x

Clone project and run test.

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ npm install
$ node src/server.js
```

* Change your admin email account in the file: **src/utils/mailer.js**
